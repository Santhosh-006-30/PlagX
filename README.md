# PlagX - AI & Plagiarism Detection System

PlagX is a comprehensive full-stack application designed to detect semantic plagiarism and AI-generated text in documents. It leverages state-of-the-art Machine Learning models to provide high-accuracy similarity scores and AI probability reports.

## 🚀 Key Features

- **Semantic Plagiarism Detection**: Uses Sentence-Transformers (SBERT) and FAISS to find matches based on meaning, not just exact word matching.
- **AI Text Detection**: Integrated RoBERTa-based classifier to detect text generated by LLMs like ChatGPT.
- **Real-time Reports**: Interactive dashboard with automatic polling for background analysis tasks.
- **Text Highlighting**: Visual document viewer that highlights specific matching segments from other sources.
- **Zero-Config Setup**: Automatically uses SQLite for easy local development without needing a separate database server.

## 🛠️ Technology Stack

- **Frontend**: Next.js 14+ (App Router), Tailwind CSS v4, Lucide Icons.
- **Backend**: FastAPI (Python 3.10+), SQLAlchemy (Async), Uvicorn.
- **Machine Learning**: 
  - `sentence-transformers/all-MiniLM-L6-v2` (for vector embeddings)
  - `Hello-SimpleAI/chatgpt-detector-roberta` (for AI detection)
  - `FAISS` (for high-speed vector similarity search)
- **Database**: SQLite (via `aiosqlite`) for development.

## 🏁 Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+

### 1. Backend Setup
```bash
cd backend
# Install dependencies (use the correct python path if needed)
pip install -r requirements.txt
# Start the server
uvicorn app.main:app --reload --port 8000
```

### 2. Frontend Setup
```bash
cd ai-plagiarism-system/frontend
# Install dependencies
npm install
# Start the UI
npm run dev
```

### 3. Usage
- Open `http://localhost:3000` in your browser.
- The system is pre-configured to bypass login for easy testing.
- Upload a PDF, DOCX, or TXT file to start the analysis.

## 📂 Project Structure
- `/backend`: FastAPI source code and ML services.
- `/ai-plagiarism-system/frontend`: Next.js application and UI components.
- `/uploads`: Temporary local storage for analyzed documents.
